# Txirrita Mendi Lasterketa ⛰️

**[EUS]** Txirrita Mendi Lasterketaren webgune ofiziala. Astro eta Tailwind CSS erabiliz eraikitako proiektu modernoa, errendimendu altua eta diseinu garbia lehenesten dituena.

**[CAS]** Sitio web oficial de la carrera de montaña Txirrita. Proyecto moderno construido con Astro y Tailwind CSS, priorizando el alto rendimiento y un diseño limpio.

---

## 🛠️ Teknologiak / Tecnologías

Este proyecto utiliza un stack tecnológico de última generación para garantizar la mejor experiencia de usuario:

*   **Framework:** [Astro](https://astro.build/) (Static Site Generation)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **Components:** Astro Components para una arquitectura modular.
*   **Interactions:** JavaScript nativo para animaciones y funcionalidades dinámicas (Countdown, Marquee, etc.).
